    #### 簡介
TrendRadar 是一個輕量、易部署的熱點助手，助您告別無效刷屏，最快30秒即可部署，只看真正關心的新聞資訊。它能聚合全網熱點，透過智慧分析與精準篩選，提供個人化新聞推送，並支援多渠道實時通知。

#### 主要功能

*   **全網熱點聚合**: 預設監控11個主流平臺（如知乎、抖音、Bilibili、百度熱搜等），並可自行增加額外平臺。
*   **智慧推送策略**: 提供 `daily` (當日彙總), `current` (當前榜單), `incremental` (增量監控) 三種模式，滿足不同使用者需求。
*   **精準內容篩選**: 透過個人關鍵字（支援普通詞、必須詞 `+`、過濾詞 `!`、數量限制 `@`）設定，只推送相關熱點。
*   **熱點趨勢分析**: 實時追蹤新聞熱度變化，記錄時間軸、熱度變化、新增檢測及跨平臺對比。
*   **個性化熱點演算法**: 根據排名高(60%)、持續出現(30%)、排名質量(10%)等權重，重新整理全網熱搜。
*   **多渠道實時推送**: 支援企業微信（含個人微信推送）、飛書、釘釘、Telegram、郵件、ntfy、Bark、Slack 等多種通知渠道。
*   **AI 智慧分析 (v3.0.0 新增)**: 基於 `MCP (Model Context Protocol)` 協議，提供對話式查詢、13種分析工具（如話題趨勢追蹤、情感分析等）。
*   **零技術門檻部署**: 支援 GitHub 一鍵 `Fork`，30秒部署 `GitHub Pages`，或透過 `Docker` 容器化部署。

#### 如何使用

*   **Fork 專案**: 點選 GitHub 頁面右上角的 "Fork" 按鈕，將專案複製到您的 GitHub 帳戶。
*   **設定 GitHub Secrets**: 在您 `Fork` 後的倉庫中，進入 `Settings > Secrets and variables > Actions > New repository secret`。
    *   **Name** 必須嚴格使用指定名稱（例如：`WEWORK_WEBHOOK_URL`、`TELEGRAM_BOT_TOKEN`、`TELEGRAM_CHAT_ID` 等）。
    *   **Secret** 填寫從對應平臺獲取的 `Webhook` 地址、`Bot Token` 或 `Chat ID` 等實際內容。
    *   可同時配置多個平臺。
*   **手動測試新聞推送**: 完成 `Secrets` 配置後，進入您 `Fork` 專案的 `Actions` 頁面，找到 "Hot News Crawler"，點選右側的 "Run workflow" 按鈕進行測試。
*   **配置推送模式**: 編輯 `config/config.yaml` 檔案中的 `report.mode` 設定，選擇 `daily` (當日彙總)、`current` (當前榜單) 或 `incremental` (增量監控) 等模式。
*   **設定個人關鍵字**: 編輯 `config/frequency_words.txt` 檔案，新增您關心的關鍵字，支援普通詞、必須詞 `+`、過濾詞 `!`、數量限制 `@` 等基礎語法。
*   **調整熱點權重**: 在 `config.yaml` 中調整 `weight` 區塊的 `rank_weight`、`frequency_weight` 和 `hotness_weight` 比例，總和必須為 1.0。
*   **Docker 部署 (可選)**: 專案提供 `Docker` 或 `docker-compose` 方式部署，您可透過環境變數覆蓋 `config.yaml` 中的配置。例如：設定 `-e REPORT_MODE="incremental"`。

#### 主要功能

*   **瀏覽器工作流程自動化：** 利用 LLMs 和 Computer Vision 自動化網頁任務，無需自訂程式碼，對網站佈局變更具有抵抗力。

*   **認證支援：** 支援多種認證方式，包括 2FA (TOTP，如 QR、Email、SMS) 和與 Bitwarden、1Password、LastPass 等密碼管理工具整合。
*   **彈性 LLM 整合：** 支援 Model Context Protocol (MCP) 和多種 LLM 供應商，如 OpenAI、Anthropic、Azure OpenAI、AWS Bedrock、Gemini、Ollama、OpenRouter 及 OpenAI-compatible 端點。

#### 如何使用

    ```python
    from skyvern import Skyvern
    skyvern = Skyvern()
    task = await skyvern.run_task(prompt="Find the top post on hackernews today")
    print(task)
    # 可指定目標 (Cloud 或本地服務): Skyvern(api_key="...", base_url="...")
    ```

    *   確保 Docker Desktop 執行。
    *   執行 `skyvern init llm` (生成 .env) 並填寫 LLM 金鑰。
    *   執行 `docker compose up -d`，然後訪問 `http://localhost:8080` 使用 UI。

    若需 X11 clipboard 相容性，需安裝 `x11-dev` 相關套件，例如 Ubuntu 上使用 `apt install xwayland`。
*   **登入方式一：Password 登入**
    直接執行 `discordo` 後，輸入您的 email 和 password 即可登入。
*   **登入方式二：Token 登入**
    透過 `--token` command-line flag 提供驗證 token (例如 `discordo --token "YOUR_DISCORD_TOKEN"`)，或設定 `DISCORDO_TOKEN` 環境變數。
*   **登入方式三：QR 登入**
    執行 `discordo` 後，點選 "Login with QR" 按鈕，並遵循螢幕上的指示操作。
*   **配置**
    應用程式的行為、keybindings 和 theme 可透過 `config.toml` 檔案進行自定義。該檔案路徑因作業系統而異 (例如 Unix 系統為 `$XDG_CONFIG_HOME/discordo/config.toml` 或 `$HOME/.config/discordo/config.toml`)。若未找到配置檔，將使用預設配置。
