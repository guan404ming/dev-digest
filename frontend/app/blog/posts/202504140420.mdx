    #### 簡介

Cursor Free VIP 是一款用於學習和研究用途的工具，目前未違反任何法律，請支援原作者。它不會生成假電子郵件帳戶或 OAuth 訪問許可權，並支援 Windows、macOS 和 Linux 系統。為獲得最佳效能，建議以管理員許可權執行並保持軟體更新。


#### 主要功能

* 支援 Windows、macOS 和 Linux 系統。
* 重設 Cursor 的設定。
* 支援多種語言：英文、簡體中文、繁體中文和越南語。
* 提供指令碼自動化執行功能 (Auto Run Script)。
* 提供手動重置機器功能 (Manual Reset Machine)。


#### 如何使用

* **指令碼自動化執行 (Auto Run Script):**
    * Linux/macOS: 使用 `curl -fsSL https://raw.githubusercontent.com/yeongpin/cursor-free-vip/main/scripts/install.sh -o install.sh && chmod +x install.sh && ./install.sh` 指令安裝。
    * Archlinux: 使用 AUR 安裝 `yay -S cursor-free-vip-git`。
    * Windows: 使用 `irm https://raw.githubusercontent.com/yeongpin/cursor-free-vip/main/scripts/install.ps1 | iex` 指令安裝。
* **手動執行重置機器 (Manual Reset Machine):**
    * Linux/macOS: 使用 `curl -fsSL https://raw.githubusercontent.com/yeongpin/cursor-free-vip/main/scripts/reset.sh | sudo bash` 指令重置。
    * Windows: 使用 `irm https://raw.githubusercontent.com/yeongpin/cursor-free-vip/main/scripts/reset.ps1 | iex` 指令重置。
* 停止指令碼請按下 Ctrl+C。
*  配置檔案位於 `Documents/.cursor-free-vip/config.ini` (Win/macOS/Linux)，需根據自身系統路徑調整。  配置檔案包含多項引數設定，例如 Chrome 路徑、等待時間等等，詳見範例配置檔案。
## 📌 [virattt/ai-hedge-fund](https://github.com/virattt/ai-hedge-fund)
<Callout>
    Description: An AI Hedge Fund Team\
    🌐 Python｜⭐️ 25,920 | 4965 stars this week
</Callout>
    #### 簡介

* 此專案是一個概念驗證 (Proof of Concept)，旨在探索使用人工智慧 (AI) 來進行交易決策的可能性。
* 系統模擬一個 AI 驅動的避險基金，並非實際進行交易或投資。
*  系統包含多個 AI 代理，分別模擬不同投資策略及風格，例如價值投資、成長投資、主動投資等。
*  僅供教育及研究用途，不保證任何投資回報，且不承擔任何財務損失責任。
*  使用者需自行承擔所有風險，並建議諮詢財務顧問以獲得專業投資建議。
*  程式使用 Python 編寫，並需安裝 Poetry 及相關套件。
*  需設定 OpenAI、Groq 或其他 LLM 提供商的 API 金鑰以及 Financial Datasets API 金鑰 (部分股票資料免 API 金鑰)。


#### 主要功能

* 模擬多位著名投資大師的投資策略，包含價值投資、成長投資、主動投資等不同風格。
*  整合多個 AI 代理，包含估值代理、情緒代理、基本面代理、技術面代理、風險管理代理和投資組合管理代理。
*  提供避險基金模擬功能，根據設定的股票程式碼及時間範圍，模擬交易決策。
*  提供回測功能 (Backtester)，評估策略在歷史資料中的表現。
* 使用 LLM (大型語言模型) 進行決策推理。
* 支援多個 LLM 提供商，例如 OpenAI 和 Groq。
*  允許使用者自訂股票程式碼、時間範圍等引數。
* 可選擇顯示每個代理的決策推理過程。


#### 如何使用

* 使用 `git clone` 命令複製專案程式碼。
* 使用 `curl` 命令安裝 Poetry 套件管理工具。
* 使用 `poetry install` 命令安裝專案相依套件。
* 建立 `.env` 檔案並設定 API 金鑰 (OpenAI, Groq, Financial Datasets)。
* 使用 `poetry run python src/main.py` 命令執行避險基金模擬，並指定股票程式碼 (例如 `--ticker AAPL,MSFT,NVDA`) 及其他選項 (例如 `--start-date`, `--end-date`, `--ollama`, `--show-reasoning`)。
* 使用 `poetry run python src/backtester.py` 命令執行回測功能，並指定股票程式碼及其他選項。
*  需注意，部分股票資料需要 API 金鑰，而 AAPL, GOOGL, MSFT, NVDA, TSLA 等股票資料則免費提供。
